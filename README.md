## What

A GUI for Zookeeper created by JavaFX and Curator Framework.

## Requires

- Java 1.8 +

## Download

[Download from Here](https://github.com/vran-dev/PrettyZoo/releases)

- usage
```shell
java -jar PrettyZoo.jar
```
